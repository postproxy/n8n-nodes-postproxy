# n8n-nodes-postproxy

Community n8n node for the PostProxy API.

PostProxy provides a unified API for publishing and scheduling posts across multiple social media platforms with built-in retries, quota handling, and per-account status tracking.

## Features (v1)
- List connected social accounts
- Publish posts to one or multiple accounts
- Schedule posts for future publishing

## Requirements
- n8n v1+
- PostProxy API key

## Installation (local)
```bash
npm install n8n-nodes-postproxy
